This is a study on A Trust-Calibrated Intent-Based Framework for Autonomous Ransomware Containment over QUIC Using Programmable Data Planes and Reinforcement Learning. It provides the background to the study by detailing why ransomware containment has become a major problem in modern encrypted and high-speed network setting.
